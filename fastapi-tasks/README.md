# Final-Project_
Metehan Cakmak - FASTAPI Task Management System:

Setup & Installation: pip install fastapi uvicorn
To Run: uvicorn main:app --reload

API available at: http://127.0.0.1:8000
Swagger UI: http://127.0.0.1:8000/docs

API Endpoints:
API Endpoint 1: Root Check
API Endpoint 2: Get Task Statistics
API Endpoint 3: Get All Tasks & Filter Tasks (for completed and not completed)
API Endpoint 4: Create Task
API Endpoint 5: Delete All Tasks
API Endpoint 6: Get Single Task
API Endpoint 7: Update Task
API Endpoint 8: Delete Task
